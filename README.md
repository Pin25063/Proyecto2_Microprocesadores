# Proyecto 02 - Zelda
---
## Descripción
Este proyecto consiste en una recreación inspirada en el videojuego clásico The Legend of Zelda,  
desarrollada en C++ para terminal utilizando la librería ncurses.

## Tecnologías utilizadas
- C++
- ncurses
- pthreads

## Compilación
Para compilar el proyecto adecuadamente, incluyendo los archivos ".hpp" y ".cpp":
g++ src/*.cpp -Iinc -lncurses -o zelda
- Ejecución: ./zelda
